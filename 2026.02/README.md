# 📘 알고리즘 핵심 개념 학습 스터디

SSAFY 교육생들의 알고리즘 문제 해결 역량 향상을 목표로  
체계적인 커리큘럼을 기반으로 개념 학습과 문제풀이를 병행하는 스터디입니다.

---

## 📌 스터디 개요

- **스터디명**: 두근두근 알고리즘 스터디
- **기간**: 2026.01.29 ~ 
- **진행 방식**: 온라인 / 오프라인 병행
- **대상**: SSAFY 교육생
- **목표**: 알고리즘 핵심 개념 체득 및 실전 적용 능력 향상

---

## 🎯 스터디 목표

본 스터디는 단계별 알고리즘 커리큘럼을 기반으로  
다음과 같은 목표를 달성하는 것을 목표로 합니다.

- 기초 수학부터 그래프, DP, 그리디까지 핵심 알고리즘 개념 학습
- 문제풀이를 통한 개념 이해 및 응용 능력 강화
- 코드 리뷰를 통한 논리력 및 구현 능력 향상
- 코딩 테스트 대비 실전 감각 향상

---

## 🧩 진행 방식

스터디는 다음과 같은 방식으로 운영됩니다.

1. 사전 개념 학습
2. 공통 문제 풀이
3. 풀이 공유 및 발표
4. 코드 리뷰 및 피드백
5. 학습 내용 정리

- 주 2회(화,목) 정기 모임 진행
- 주차별 담당자 발표 운영
- 참여도 및 출석 관리

---

## 📚 커리큘럼

| 주차 | 주제 | 핵심 내용 |
|------|------|-----------|
| 1~2주 | 수학 기초 | GCD/LCM, 소수 판별, 에라토스테네스, 골드바흐의 추측 |
| 3~4주 | 기초 자료구조 | 스택, 큐, 덱, 힙, 딕셔너리 |
| 5~6주 | 재귀 & 완전탐색 | 순열, 조합, 브루트포스 |
| 7~8주 | DP | 점화식, 메모이제이션 |
| 9~10주 | 그래프 탐색 | DFS, BFS |
| 11~12주 | 최단경로 | 다익스트라, 플로이드 |
| 13~14주 | 이분탐색 & 분할정복 | 파라메트릭 서치 |
| 15~16주 | 그리디 | 스케줄링, 최소 비용 |
| 17~18주 | 시뮬레이션 | 상태 관리 |
| 19~20주 | 문자열 & 백트래킹 | 슬라이딩 윈도우, N-Queen |
