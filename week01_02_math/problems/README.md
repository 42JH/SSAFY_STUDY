## 1. 베르트랑 공준

문제:  
베르트랑 공준은 임의의 자연수 n에 대하여, n보다 크고 2n보다 작거나 같은 소수가 적어도 하나 존재한다는 내용이다. 자연수 n이 주어졌을 때, n보다 크고 2n보다 작거나 같은 소수의 개수를 구하라.

입력:  
여러 테스트 케이스로 이루어져 있으며 각 줄에 n이 주어진다. 마지막 줄에는 0이 주어진다.

출력:  
각 테스트 케이스마다 n보다 크고 2n보다 작거나 같은 소수의 개수를 출력한다.

제한:  
1 ≤ n ≤ 123,456

예제 입력:
```
1
10
13
100
1000
10000
100000
0
```

예제 출력:
```
1
4
3
21
135
1033
8392
```

## 2. 골드바흐 파티션

문제:  
짝수 N(2 < N ≤ 1,000,000)을 두 소수의 합으로 표현하는 방법의 수(순서만 다른 것은 동일)를 구하라.

입력:  
첫 줄에 테스트 케이스 수 T(1 ≤ T ≤ 100). 이어서 각 줄에 짝수 N이 주어진다.

출력:  
각 테스트 케이스마다 골드바흐 파티션의 수를 출력한다.

예제 입력:
```
5
6
8
10
12
100
```

예제 출력:
```
1
1
2
1
6
```

## 3. 숨바꼭질 6

문제:  
수빈이의 위치 S와 N명의 동생 위치 Ai가 주어질 때, 모든 동생을 찾기 위해 정할 수 있는 정수 D의 가능한 최댓값을 구하라. (수빈이는 한 걸음에 X±D 이동)

입력:  
첫 줄에 N(1 ≤ N ≤ 10^5)과 S(1 ≤ S ≤ 10^9). 둘째 줄에 N개의 Ai(1 ≤ Ai ≤ 10^9, 서로 다름, S와 같지 않음).

출력:  
가능한 D의 최댓값을 출력한다.

예제 1 입력:
```
3 3
1 7 11
```
예제 1 출력:
```
2
```

예제 2 입력:
```
3 81
33 105 57
```
예제 2 출력:
```
24
```

예제 3 입력:
```
1 1
1000000000
```
예제 3 출력:
```
999999999
```

추가할 때 형식은
"문제명 - 이름"
형식으로 작성
