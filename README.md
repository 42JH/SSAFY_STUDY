| **주차** | **주제** | **핵심 내용** |
| --- | --- | --- |
| **1~2주** | **수학 기초** | 최대공약수/최소공배수, 소수 판별, 에라토스테네스, 약수, 조합 기초 |
| **3~4주** | **기초 자료구조** | 스택, 큐, 덱, 힙, 딕셔너리/집합 활용 |
| **5~6주** | **재귀 & 완전탐색** | 재귀 구조, 순열, 조합, 부분집합, 브루트포스 |
| **7~8주** | **DP (동적 계획법)** | 1·2차원 DP, 점화식 설계, 메모이제이션 |
| **9~10주** | **그래프 탐색 (DFS/BFS)** | 인접리스트, 방문배열, 미로 탐색, 연결 요소 |
| **11~12주** | **최단경로** | BFS 최단거리, 다익스트라, 플로이드 워셜 |
| **13~14주** | **이분탐색 & 분할정복** | 이분탐색, 파라메트릭 서치, 병합정렬, 거듭제곱 분할 |
| **15~16주** | **그리디** | 정렬 기반 선택, 활동 선택, 스케줄링, 최소 비용 |
| **17~18주** | **완전탐색 & 시뮬 & 구현** | 배열 조작, 격자 시뮬, 삼성 유형, 상태 관리 |
| **19~20주** | **문자열 & 투포인터 & 백트래킹** | 문자열 처리, 슬라이딩 윈도우, N-Queen, 가지치기 |
